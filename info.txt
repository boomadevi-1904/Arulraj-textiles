Project: Arulraj Textiles (MERN)

Overview
- Full-stack MERN application for showcasing textile collections, ordering, and tracking.
- Elegant theme with custom cloth-like cursor and responsive UI.
- Admin can manage products and update order statuses.

Technology Stack
- Client: React + Vite, React Router, Tailwind CSS, Framer Motion
- Server: Node.js, Express.js, Mongoose, JWT
- DB: MongoDB Atlas

Environment
- Server .env (already set): 
  MONGO_URI=mongodb+srv://user:user@cluster0.wqniobg.mongodb.net/?appName=Cluster0
  NODE_ENV=development
  PORT=5000
  JWT_SECRET=any_secure_secret

Run Locally
- Server: cd server && npm install && node index.js
- Client: cd client && npm install && npm run dev

Customer Flow
1) Add to cart from Shop/Product pages
2) Go to Cart → Proceed to Checkout
3) Login/Register
4) Fill Shipping details (saved in localStorage)
5) Review in Place Order and submit (POST /api/orders)
6) See order ID and track via Track Order
7) Profile shows account info and order history

Server API (Protected with JWT)
- Users:
  POST /api/users        (register)
  POST /api/users/login  (login)
  GET  /api/users/profile (me)
- Products:
  GET  /api/products
  GET  /api/products/:id
- Orders:
  POST /api/orders             (create)
  GET  /api/orders/myorders    (my list)
  GET  /api/orders/:id         (details)
  PUT  /api/orders/:id/pay     (mark paid)
  PUT  /api/orders/:id/deliver (admin deliver)

Admin Features
- Login with admin credentials
- Post and manage products
- View and update order status (delivered)
